# mindform Active — Assets Guide

## Estructura de carpetas

```
assets/
  logo/              ← Logo de la marca (PNG transparente recomendado)
  images/
    hero/            ← Fotos grandes para secciones hero (mínimo 1920px ancho)
    products/        ← Fotos de productos (relación 3:4 recomendada)
    lifestyle/       ← Fotos de lifestyle / editorial / brand story
    categories/      ← Fotos para el grid de categorías (Tops, Leggings, Sets, Accessories)
```

## Logo

Poner el logo aquí:
→ `assets/logo/logo.png`  (versión oscura, sobre fondo claro)
→ `assets/logo/logo-light.png`  (versión clara, sobre fondo oscuro / hero)

## Recomendaciones de imágenes

| Carpeta      | Uso                        | Tamaño ideal     | Formato |
|--------------|----------------------------|------------------|---------|
| hero/        | Fondos de sección hero     | 1920 × 1080 px+  | JPG     |
| products/    | Fotos de producto          | 800 × 1067 px    | JPG     |
| lifestyle/   | Brand story, CTA sections  | 1200 × 800 px+   | JPG     |
| categories/  | Grid de 4 categorías       | 600 × 800 px     | JPG     |
